Seguimientos / Tracker App

Es una App de interacción móvil que se ejecuta bajo la plataforma Android que es utilizado para dar seguimiento por geolocalización. Con parametrización del dispositivo para gestionar la actividad de este, cuenta con un menú de Parámetro, Presentación, Mensaje y Mapa. Esta aplicación creada por Android para el lado del cliente y luego PHP MySQL para el lado de administración. Corre bajo Plataforma Android que es el sistema operativo más popular del mundo.
